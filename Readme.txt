This Python virtual personal assistant is a Voice Activated Desktop Assistant and hence named as "Python-Virtual-Voice-Activated-Personal-Desktop-Assistant"

This Voice Activated Desktop Assistant will take microphone input from the user and performs following tasks :-
Task 1 : To search something on Wikipedia
Task 2 : To open YouTube site in browser
Task 3 : To open Google site in browser
Task 4 : To play music 
Task 5 : To know current time
Task 6 : To open VS Code Program
Task 7 : To send E-mail
Task 8 : Quit or Exit 

Install dependencies :- pyttsx3
                        SpeechRecognition 
                        datetime
                        wikipedia
                        webbrowser
                        os
                        smtplib
